# OSINT Investigations

Open Source Intelligence (OSINT) investigations to track and uncover cyber threats.

## Investigation Examples

### Phishing Domain Network
Mapping domains and subdomains used in global phishing campaigns.

### Social Media Threat Actor Tracking
Following threat actors and groups across platforms to analyze campaigns.

### Infrastructure Mapping
Identifying servers, hosting providers, and IP blocks used in cyber attacks.

## Tools & Techniques

- Maltego & SpiderFoot for intelligence mapping  
- WHOIS & DNS lookup for domain investigations  
- Shodan & Censys for exposed services  
- Python automation scripts for data collection
