# Tools & Frameworks

This section presents tools and frameworks used for cyber threat intelligence and analysis.

## Featured Tool: Hunter X

**Hunter X** is a professional framework for:

- Threat intelligence gathering  
- Malicious link detection  
- OSINT automation  
- Malware and infrastructure tracking

## Additional Tools

- **Python Scripts**: Automate phishing detection, URL analysis, and IOC extraction.  
- **Kali Linux Tools**: nmap, Wireshark, netcat, and more for network & malware analysis.  
- **Maltego**: Graph-based intelligence and link analysis.  
- **Shodan**: Scan and analyze exposed IoT devices and servers.  

Each tool and script is aimed at **enhancing the detection and understanding of cyber threats**, helping analysts and organizations strengthen their defenses.
